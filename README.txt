1. Extract all base station folders into a folder named 'CORS_stations'. It must be in the same folder as the 'main.py' file.
2. Run the 'create_csvs.py'. This will create a folder called CSVs and create csv files for the respective stations.
3. Run the 'main.py' and wait till execution is done, it may take a while.
4. Then open the CSVs folder and inspect your extracted data.
